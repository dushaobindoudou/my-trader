# 交易模块页面设计

## 页面结构

### 1. 交易主页面 (`/trades`)
- **功能**：查看市场数据、交易历史、持仓
- **组件**：
  - K线图表（支持时间周期切换）
  - 交易对选择器
  - 交易历史列表（Tab）
  - 持仓列表（Tab）
- **跳转**：
  - 点击"新建交易" → `/trades/new`
  - 点击交易记录 → `/trades/[id]`
  - 点击持仓 → `/portfolio`

### 2. 创建交易页面 (`/trades/new`)
- **功能**：创建新的交易订单
- **组件**：
  - 交易表单（交易对、方向、类型、数量、价格）
  - 实时价格显示
  - 风险提示
- **跳转**：
  - 提交后 → `/trades`（显示新订单）
  - 取消 → `/trades`

### 3. 交易详情页面 (`/trades/[id]`)
- **功能**：查看单个交易的详细信息
- **组件**：
  - 交易基本信息
  - 订单状态
  - 执行历史
  - 相关图表
- **跳转**：
  - 返回 → `/trades`
  - 取消订单 → `/trades`

### 4. 交易历史页面 (`/history`)
- **功能**：查看所有交易历史记录
- **组件**：
  - 交易历史列表（支持筛选、排序、分页）
  - 统计信息（总交易数、盈亏等）
  - 筛选器（交易对、时间范围、状态等）
- **跳转**：
  - 点击交易记录 → `/trades/[id]`

### 5. 投资组合页面 (`/portfolio`)
- **功能**：查看投资组合总览
- **组件**：
  - 账户总览（总资产、可用余额、盈亏等）
  - 持仓列表
  - 资产分布图表
  - 盈亏统计
- **跳转**：
  - 点击持仓 → `/trades`（筛选该交易对）
  - 点击交易 → `/trades/[id]`

### 6. 观察区页面 (`/watchlist`)
- **功能**：管理关注的交易对
- **组件**：
  - 观察列表
  - 价格提醒设置
  - 快速交易入口
- **跳转**：
  - 点击交易对 → `/trades?symbol=XXX`

## 页面跳转关系图

```
首页 (/)
  ├─ 交易 (/trades)
  │   ├─ 新建交易 (/trades/new)
  │   └─ 交易详情 (/trades/[id])
  │
  ├─ 交易历史 (/history)
  │   └─ 交易详情 (/trades/[id])
  │
  ├─ 投资组合 (/portfolio)
  │   ├─ 交易 (/trades?symbol=XXX)
  │   └─ 交易详情 (/trades/[id])
  │
  └─ 观察区 (/watchlist)
      └─ 交易 (/trades?symbol=XXX)
```
