# 交易模块页面实施总结

## 已创建的页面

### 1. 交易主页面 (`/trades`)
**文件**: `src/app/trades/page.tsx`

**功能**:
- 显示市场数据区域（K线图表占位）
- 交易历史和持仓的 Tab 切换
- "新建交易"按钮跳转到 `/trades/new`

**跳转关系**:
- 点击"新建交易" → `/trades/new`
- 点击交易记录 → `/trades/[id]`（待集成）

**状态**: ✅ 基础框架完成，待集成 API

---

### 2. 创建交易页面 (`/trades/new`)
**文件**: `src/app/trades/new/page.tsx`

**功能**:
- 交易表单（交易对、方向、类型、数量、价格）
- 实时价格显示（占位）
- 风险提示
- 提交和取消按钮

**跳转关系**:
- 点击"取消" → 返回上一页
- 点击"提交订单" → `/trades`（待集成）

**状态**: ✅ 基础框架完成，待集成 API

---

### 3. 交易详情页面 (`/trades/[id]`)
**文件**: `src/app/trades/[id]/page.tsx`

**功能**:

- 订单状态显示
- 时间信息
- 价格走势图表（占位）
- 取消订单按钮（仅待成交订单）

**跳转关系**:
- 点击返回按钮 → 返回上一页
- 点击"取消订单" → `/trades`（待集成）

**状态**: ✅ 基础框架完成，待集成 API

### 4. 交易历史页面 (`/history`)

**状态**: ✅ 基础框架完成，待集成 API

### 5. 投资组合页面 (`/portfolio`)

**跳转关系**:

## 页面跳转关系图

```
首页 (/)
  │
  ├─ 交易 (/trades)
  │   ├─ 新建交易 (/trades/new)
  │   │   └─ 提交后 → /trades
  │   │
  │   └─ 交易详情 (/trades/[id])
  │       └─ 返回 → /trades
  │
  ├─ 交易历史 (/history)
  │   ├─ 点击交易 → /trades/[id]
  │   └─ 创建交易 → /trades/new
  │
  ├─ 投资组合 (/portfolio)
  │   ├─ 点击持仓 → /trades?symbol=XXX
  │   ├─ 查看全部 → /trades
  │   └─ 创建交易 → /trades/new
  │
  └─ 观察区 (/watchlist)
      ├─ 点击观察项 → /trades?symbol=XXX
      ├─ 快速交易 → /trades/new?symbol=XXX
      └─ 添加标的 → 对话框（待实现）
```
